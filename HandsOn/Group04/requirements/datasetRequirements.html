Dataset Requirements - Grupo 04

Dataset: Alquiler medio por barrios (Madrid)

Este documento analiza si el dataset seleccionado cumple los requisitos establecidos para la práctica Hands-on 1.

Requisito Descripción Análisis Conclusión
R1 Los datos pertenecen a un dominio de ciudad inteligente (Smart City). El dataset recoge información sobre el precio del alquiler por metro cuadrado, número de viviendas, dormitorios y baños en diferentes barrios de Madrid. Este tipo de información pertenece al dominio de vivienda y planificación urbana, uno de los pilares de las Smart Cities. Cumple
R2 Los datos están disponibles en formato CSV. El dataset se encuentra en formato CSV, con campos separados por punto y coma (;). Esto facilita su tratamiento y posterior conversión a RDF. Cumple
R3 Los datos tienen una licencia abierta que permite su publicación. El dataset procede del portal de datos abiertos del Ayuntamiento de Madrid (datos.madrid.es), donde los conjuntos de datos se publican bajo licencia abierta que permite su reutilización y redistribución. Cumple
R4 Los datos pueden vincularse fácilmente con entidades reales (por ejemplo, ubicaciones). El dataset incluye el nombre del barrio, que puede enlazarse con entidades geográficas existentes como Wikidata o DBpedia (por ejemplo, “Barrio de Salamanca”). Esto facilita la vinculación semántica. Cumple
R5 (opcional) Existe documentación para los datos. El portal de datos proporciona una descripción general y el significado de cada campo, pero no incluye un diccionario técnico detallado. Por tanto, la documentación es parcial. Parcialmente cumple
R6 (opcional) Los datos pueden provenir de múltiples fuentes. Actualmente, los datos provienen de una única fuente (Ayuntamiento de Madrid). Sin embargo, podrían combinarse con datos de otros portales (por ejemplo, INE o Ministerio de Vivienda) para ampliar el contexto. No cumple (aunque sería posible ampliarlo)

Conclusión: El dataset cumple todos los requisitos obligatorios (R1–R4) y parcialmente los opcionales (R5–R6), por lo que es adecuado para su uso en las siguientes fases del proyecto.
